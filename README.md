# PhonePe Data Analysis Dashboard (Power BI)
End-to-end PhonePe data analysis project created using Power BI.

This project is an end-to-end data analysis and visualization of PhonePe digital payment trends.  
The dashboard was created using Power BI by following a guided tutorial and analyzing the PhonePe dataset.

## Tools Used
- Power BI Desktop
- Power Query
- DAX Measures
- Excel/CSV dataset from tutorial

## What the Project Includes
- Data import & cleaning
- Data modeling & DAX formulas
- Visualizations (Maps, charts, KPIs)
- State-wise and category-wise analysis
- Fully interactive dashboard

## How to View
Download the `.pbix` file and open it in Power BI Desktop.
